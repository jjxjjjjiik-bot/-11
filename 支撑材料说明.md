# 支撑材料说明

本支撑材料用于复现论文四个问题的输入、计算结果和图表，不包含承诺书、编号页或身份信息。

## 运行入口

- 问题一：`matlab/q1/main_question1.m`
- 问题二：`matlab/q2/run_q2_all.m`
- 问题三：`matlab/q3/run_q3_all.m`
- 问题四：`matlab/q4/run_q4_all.m`
- 问题四人口输入提取：`matlab/q4/prepare_q4_inputs.py`

各问题的输入、输出位于对应目录，主程序文件头给出调用顺序。`figures/` 保存论文使用及结果复核所需图片，`数据来源/` 保存来源清单、提取记录和可提交的最小数据文件。

## 文件边界

- 不包含仅用于开发阶段独立验证的 Python 脚本。
- WPP 2024 原始压缩 CSV 和大型 Excel 合计超过 20 MB，不放入压缩包；保留 2024--2050 年最小人口输入、确定性提取程序、官方 URL 和 SHA-256。
- 问题三的大型官方指南 PDF 不重复打包；来源清单保留官方链接，本包提供参数表、提取记录和文字证据说明。
