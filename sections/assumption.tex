\section{模型假设}

\subsection{问题一的模型假设}

为在5个稀疏观测点下保持模型可识别和结论可解释，作如下假设：

\begin{enumerate}[label=(\arabic*),leftmargin=2em]
  \item 五个已核实数据点使用一致的成人超重肥胖合计口径，能够反映我国总体变化趋势。
  \item 观测年份之间不进行插值，Logistic和Gompertz模型直接以实际年份为自变量处理非等间距数据。
  \item 成人超重肥胖率位于$0$--$100\%$之间，主分析将$K=100\%$作为数学物理上限，而不把它解释为未来一定达到的实际平台值。
  \item 在研究时段内不存在足以使历史增长机制完全突变且无法由情景分析表达的外生冲击。
  \item 模型只预测全国成人超重肥胖合计比例，不将结果直接拆分为超重率、肥胖率或具体慢性病发生率。
\end{enumerate}

\subsection{问题二的模型假设}

为统一不同研究的证据口径并保证排序可解释性，作如下假设：

\begin{enumerate}[label=(\arabic*),leftmargin=2em]
  \item 系统综述纳入的研究能够代表非手术、非药物成人减重维持的一般证据，用于限定模型的适用人群和干预范围。
  \item 研究数量、方向一致率和证据等级均按“数值越大，证据越重要”处理，使三个指标统一为效益型指标。
  \item 同一综述中的Strong与Moderate证据等级可分别编码为2和1，用有序数值近似表示证据强弱。
  \item 原始表格中的正、负方向均已统一为“缺少该因素会增加减重维持失败风险”，从而保证因素方向一致。
  \item 补充证据与15项因素的结局或统计口径不同，只用于解释失败机制，不进入CRITIC-TOPSIS排序。
  \item TOPSIS得分仅反映因素的相对证据优先级，不直接等同于因果效应大小或个体减肥失败概率。
\end{enumerate}

\subsection{问题三的模型假设}

为保证画像、方案和排序结果具有统一边界，作如下假设：

\begin{enumerate}[label=(\arabic*),leftmargin=2em]
  \item 五类生活方式画像用于表示典型工作生活情境，不等同于对真实个体的医学诊断，也不假设其来自统计聚类。
  \item 候选方案均面向可开展生活方式干预的成人超重肥胖人群；存在严重疾病、妊娠、药物或手术适应证时，应由专业人员另行评估。
  \item 每日膳食能量摄入削减量、每周有氧时间、抗阻次数和睡眠目标构成排序前的指南范围初筛，任一条件不满足的方案不进入TOPSIS评价；该初筛不替代基于性别、基础能量需要量和实际摄入量的个体安全评估。
  \item 画像风险采用低、中、高、极高四档，失败因素覆盖度采用未覆盖、间接覆盖、直接覆盖三档，使专家规则保持离散、公开和可复核。
  \item 90\,kg基准体重、7700\,kcal/kg和6个月减重估计仅用于候选方案的相对比较，不视为个体临床减重预测。
  \item TOPSIS得分只在同一画像的3套候选方案内具有比较意义，不跨画像比较，也不解释为减重成功率。
\end{enumerate}

\subsection{问题四的模型假设}

为区分人口预测、模型结果与政策情景，作如下假设：

\begin{enumerate}[label=(\arabic*),leftmargin=2em]
  \item 联合国WPP 2024中国中方案的单岁年中人口能够用于描述2024---2050年18岁及以上人口规模；其属于人口预测，不视为未来真实观测。
  \item 问题一成人超重肥胖率与18岁以上人口具有近似一致的年龄口径；基准情景表示不叠加本文新增干预效果的反事实路径，不是未来必然结果。
  \item 覆盖率、条件依从率和效果均位于$[0,1]$，依从率以已覆盖人群为条件，三项相乘后得到综合实现效果。
  \item 低、中、高参数和三角分布属于本文情景假设，不代表全国政策已经达到相应水平；模拟中暂假设覆盖、依从和效果相互独立。
  \item Logistic/Gompertz等权抽样和$K$的三角分布只用于扩展联合情景，不解释为模型结构与饱和上限的客观概率分布。
  \item 可避免人数表示某一年相对基准减少的成人超重肥胖现患人数，不视为当年治愈人数，也不直接换算为慢病病例、医疗费用或经济损失。
\end{enumerate}
